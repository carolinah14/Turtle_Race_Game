# Turtle_Race_Game

A Python Turtle graphics simulation where multiple turtles race across the screen. The program uses object modelling, loops, randomness, and user input to simulate a race and determine a winner based on real-time movement.

# ▶️ How to Run

1. Make sure you have Python installed on your computer.

2. Download or clone this repository.

3. Ensure the file is in your folder:

    turtle_race_game.py

4. Open a terminal (or your preferred Python IDE).

5. Navigate to the folder containing the file.

6. Run the script using:

    python turtle_race_game.py

7. A Turtle graphics window will open. Enter your bet in the prompt and watch the race.

# 🧠 Concepts Practiced

- Using loops to control continuous simulation
- Working with multiple class instances (Turtle objects)
- Managing object state (e.g., position and color)
- Handling user input with `screen.textinput()`
- Using randomness with `random.randint()`
- Controlling program flow with boolean flags
- Using `break` to stop loops when a condition is met
- Positioning objects using coordinates (x, y)
- Separating setup logic from game loop logic
- Implementing basic game logic (win condition and user feedback)
